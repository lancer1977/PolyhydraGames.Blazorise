# Project Atlas

## Purpose

`Core.Blazorise` provides reusable Blazor component helpers, dialog shims, current-page state, reactive helpers, and query-string binding utilities for PolyhydraGames UI projects.

## Primary Validation

```bash
dotnet test PolyhydraGames.BlazorComponents.sln --configuration Release --verbosity minimal
```

## Package Surface

- Main project: `src/PolyhydraGames.BlazorComponents.csproj`
- Tests: `tests/BlazorComponents.Tests`
- Package artifact: `artifacts/package`
